CSV(Comma-Seperated Values)

CSV 특징

  1. 한 줄에 데이터 하나
  2. 첫 번쨰 줄은 헤더로 사용 가능

SSV: 뛰어쓰기
TSV: tab
CSV > TSV, SSV

xml 글자 많음(데이터 많음) > json > csv

표현력 많음: xml > json > csv

xml은 잘 쓰이지 않고 있다고함
